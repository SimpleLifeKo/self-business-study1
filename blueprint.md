
# Lotto Number Generator

## Overview

A simple web application to generate 5-digit lottery numbers.

## Project Outline

*   **`index.html`**: Contains the structure of the page, including a button to generate the numbers and a container to display them.
*   **`style.css`**: Styles the page for a clean and modern look.
*   **`main.js`**: Contains the JavaScript logic to generate and display the random lottery numbers.
*   **Disqus Comments**: Integrated into `index.html` for user comments. **Important:** Remember to replace `YOUR_SHORTNAME` in `index.html` with your actual Disqus shortname.

## Current Plan

1.  **`index.html`**: Basic HTML structure created, Disqus comments integrated.
2.  **`style.css`**: Add styles for the layout and components.
3.  **`main.js`**: Implement the number generation logic.
4.  **Deploy**: Deploy the application to Firebase Hosting.
